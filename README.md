# Image-Based Toggle Switch

A visually appealing Toggle Switch component using images for On/Off states. Perfect for showcasing UI design in web projects.

![Toggle Switch Preview1](images/day.png)
![Toggle Switch Preview2](images/night.png)

## Features

- Beautiful, image-based design  
- Smooth toggle animation  
- Fully visual, no code required to showcase  
- Perfect for UI demos and project portfolios  

## How to Use

Simply open the images or embed them in your project for demonstration purposes. No coding needed for the showroom display.
